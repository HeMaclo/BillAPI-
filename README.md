# 🎯 BillAPI - Мод для Minecraft Fabric

**BillAPI** - это мощный и удобный мод для Minecraft 1.21.1, разработанный на платформе **Fabric**. Мод включает три основные фичи для улучшения игрового опыта: **Autoclicky**, **AimAssist** и **ESP**.

> **⚠️ Внимание:** Этот мод предназначен исключительно для развлечения и обучения. Использование на серверах может быть запрещено администрацией. Используйте с умом! 😄

---

## ✨ Возможности

### 🖱️ **Autoclicky**
Автоматический удар по мобам и игрокам при наведении на них.

**Возможности:**
- ✅ Два режима клика: **1.8 PvP** и **1.9+ PvP**
- ✅ Настраиваемая задержка между ударами (1-20 тиков)
- ✅ Отдельное управление левым и правым кликом
- ✅ Автоматическое срабатывание при наведении на хитбокс
- 🎮 Клавиша по умолчанию: **V**

**Параметры:**
| Параметр | Значение | Описание |
|----------|----------|---------|
| Режим | 1.8 / 1.9+ | Режим клика (1.8 - быстрый, 1.9+ - с задержками) |
| Задержка | 1-20 | Задержка между ударами (в тиках) |
| ЛМК | Вкл/Выкл | Левый клик (удар) |
| ПМК | Вкл/Выкл | Правый клик (интеракция) |

---

### 🎯 **AimAssist**
Плавный и легитный наводсок на игроков для улучшения точности.

**Возможности:**
- ✅ Автоматический поиск ближайшего игрока
- ✅ Плавный и естественный наводсок на центр головы
- ✅ Настраиваемая чувствительность (0.0 - 1.0)
- ✅ Регулировка скорости наводсока
- ✅ Максимальная дистанция срабатывания
- ✅ Режим "удержание" кнопки для контроля
- 🎮 Клавиша включения: **C**
- 🎮 Клавиша удержания: **ALT**

**Параметры:**
| Параметр | Значение | Описание |
|----------|----------|---------|
| Чувствительность | 0.0-1.0 | Скорость наводсока (1.0 = мгновенный) |
| Скорость | 1-20 | Длительность плавного наводсока (тики) |
| Дистанция | 1-64 | Максимальная дистанция поиска игроков (блоки) |
| Плавность | Вкл/Выкл | Плавный наводсок вместо мгновенного |

---

### 👻 **ESP**
Визуальное выделение игроков и мобов сквозь стены.

**Возможности:**
- ✅ Выделение игроков
- ✅ Выделение враждебных мобов
- ✅ Выделение животных (опционально)
- ✅ Настраиваемый цвет боксов
- ✅ Отрисовка сквозь стены (без блокирования вида)
- 🎮 Клавиша включения: **E**

**Параметры:**
| Параметр | Значение | Описание |
|----------|----------|---------|
| Показывать игроков | Вкл/Выкл | Выделять других игроков |
| Показывать мобов | Вкл/Выкл | Выделять враждебных мобов |
| Показывать животных | Вкл/Выкл | Выделять животных |
| Цвет боксов | RGB | Цвет выделения (зелёный по умолчанию) |

## 🚀 Установка

### Требования
- **Minecraft Java Edition** версии **1.21.1**
- **Fabric Loader** версии **0.15.11+**
- **Java 21+** на вашем компьютере

### Шаги установки

1. **Скачайте Fabric Installer** с [fabric.net](https://fabricmc.net/)
2. **Установите Fabric** для версии 1.21.1
3. **Скачайте BillAPI jar файл** из [Releases](https://github.com/HeMaclo/BillAPI-/releases)
4. **Положите файл** в папку `.minecraft/mods/`
5. **Установите Mod Menu** (опционально, но рекомендуется):
   - Скачайте с [CurseForge](https://www.curseforge.com/minecraft/mods/modmenu)
   - Положите в папку `.minecraft/mods/`
6. **Запустите Minecraft** и наслаждайтесь! 🎮

## ⚙️ Конфигурация

### Открытие меню настроек

**Способ 1: Через Mod Menu (рекомендуется)**
1. На главном экране нажмите **"Mods"**
2. Найдите **"BillAPI"** в списке
3. Нажмите кнопку конфига (обычно рядом с названием мода)
4. Откроется меню настроек

**Способ 2: Горячие клавиши**
- Нажимайте **V**, **C**, **E** для быстрого включения/отключения фич
- Удерживайте **ALT** для использования AimAssist в режиме удержания

### Расположение файла конфига

Конфиг сохраняется в:
```
%appdata%/.minecraft/config/billapi/settings.json
```

Вы можете отредактировать его вручную, но рекомендуется использовать встроенное меню.

## 🎮 Управление

### Клавиши по умолчанию

| Фича | Клавиша | Действие |
|------|---------|---------|
| **AutoClicky** | V | Включить/выключить |
| **AimAssist** | C | Включить/выключить |
| **AimAssist Hold** | ALT | Удерживать для использования |
| **ESP** | E | Включить/выключить |

Все клавиши можно переназначить в **Управлении** → **Управление сочетаниями клавиш**.

## 🛠️ Разработка

### Сборка проекта

```bash
# Клонируем репозиторий
git clone https://github.com/HeMaclo/BillAPI-.git
cd BillAPI-

# Собираем мод
./gradlew build

# Готовый jar находится в build/libs/
```

### Требования для разработки
- **Java 21+** JDK
- **Gradle 8.0+**
- **IDE** (рекомендуется IntelliJ IDEA или VSCode)

### Структура кода

Весь код написан на **чистой Java 21** без использования Kotlin. Каждый класс содержит:
- ✅ Понятные и короткие комменты с юмором
- ✅ Правильную структуру и паттерны проектирования
- ✅ JavaDoc для публичных методов
